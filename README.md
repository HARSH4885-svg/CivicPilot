## 📌 Overview

**CivicPilot** is an AI-powered civic intelligence platform built for the **Google AI Studio Hackathon**.

Unlike conventional complaint portals that simply record issues, CivicPilot transforms citizen reports into structured, verified, and actionable intelligence using **Google Gemini 2.5 Flash**.

Citizens can report issues using **images, voice, or text**, while municipal authorities receive AI-prioritized, community-verified, and government-ready case files through an intelligent operational dashboard.

## 🤖 AI Intelligence

### Mission Control

CivicPilot introduces **Mission Control**, a transparent multi-agent reasoning system where six specialized AI agents collaborate to analyze and process civic issues.

* 👁 Vision Agent
* 📍 Geo Agent
* ✅ Verification Agent
* ⚡ Priority Agent
* 🛠 Resolution Agent
* 🚚 Deployment Agent

Each agent contributes specialized reasoning, making AI decisions transparent and explainable.
